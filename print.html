Print Export Instructions

  1. Wait for map to fully load
  2. Use browser's "Print" (Cmd/Ctrl+P)
  3. Select "Save as PDF"
  4. Set paper size to "Tabloid" or custom 24×36"
  5. Set orientation to "Landscape"
  6. Enable "Background graphics"
  7. For high-res: Use screenshot tool at 3x zoom, then scale in Photoshop/GIMP
